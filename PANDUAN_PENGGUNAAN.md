# 📖 Panduan Penggunaan AI Visual Studio

## 🎯 Tujuan Aplikasi

Aplikasi ini dirancang untuk **menghasilkan prompt JSON** yang dapat digunakan di berbagai AI Image Generator (Grok, ChatGPT, Gemini, Midjourney, dll) untuk membuat foto produk promosi berkualitas tinggi untuk **affiliate marketing dan UMKM**.

---

## 🚀 Cara Menggunakan

### 1️⃣ Login
- Buka aplikasi di browser
- Masukkan email yang terdaftar
- Klik "Masuk ke Dashboard"

### 2️⃣ Pilih Mode Generate

Ada 3 mode yang tersedia:

#### 📸 **Product Photoshoot**
Untuk foto produk profesional dengan berbagai latar belakang

#### 🤲 **POV Tangan**
Untuk foto dengan sudut pandang tangan yang memegang produk

#### 🎨 **Gabungkan Gambar**
Untuk menggabungkan beberapa elemen produk menjadi satu komposisi

---

## 📋 Langkah-langkah Detail

### Step 1: Upload Gambar Produk
1. Klik area upload atau drag & drop gambar
2. Pilih foto produk Anda (format: JPG, PNG)
3. Preview akan muncul di bawah area upload

### Step 2: Atur Preferensi

#### **Lighting (Pencahayaan)**
- **Light**: Pencahayaan terang dan cerah
- **Dark**: Pencahayaan dramatis dan gelap

#### **Atmosphere (Suasana)**
- **Clean**: Latar belakang bersih dan minimalis
- **Crowd**: Latar belakang dengan elemen tambahan

#### **Aspect Ratio (Rasio)**
- **1:1** (Square): Untuk Instagram post
- **16:9** (Landscape): Untuk banner/website
- **9:16** (Portrait): Untuk Instagram story/TikTok

#### **Opsi Tambahan**
- ✅ **Sertakan Logo Brand**: Jika ingin logo tetap terlihat
- ✅ **Hapus Teks Asli**: Untuk menghilangkan teks dari gambar original

### Step 3: Tambahkan Instruksi Custom (Opsional)
Ketik instruksi tambahan di textarea, contoh:
- "Tambahkan efek bokeh lembut"
- "Buat suasana sunset dengan warna hangat"
- "Fokus pada detail tekstur produk"

### Step 4: Generate!
Klik tombol **"Generate Prompt"**

## 📋 Cara Menggunakan Prompt JSON

### Di Grok (X.AI)
1. Copy prompt JSON dari salah satu variasi
2. Buka Grok di X.com
3. Paste prompt JSON
4. Upload gambar produk Anda sebagai referensi
5. Klik generate

### Di ChatGPT (OpenAI)
1. Copy prompt JSON
2. Buka ChatGPT (versi Plus/Pro dengan DALL-E)
3. Paste prompt JSON
4. Upload gambar produk sebagai referensi
5. Ketik: "Generate image based on this prompt"

### Di Gemini (Google)
1. Copy prompt JSON
2. Buka Google Gemini
3. Upload gambar produk
4. Paste prompt JSON
5. Minta Gemini untuk generate image

### Di Midjourney
1. Copy bagian "core_prompt" dari JSON
2. Buka Midjourney Discord
3. Gunakan command: `/imagine prompt: [paste core_prompt]`
4. Tambahkan parameter: `--ar 1:1 --quality 2 --stylize 1000`

### Di Stable Diffusion / DALL-E
1. Copy "core_prompt" dari JSON
2. Paste ke prompt box
3. Set resolution ke 4K atau maksimal yang tersedia
4. Adjust settings sesuai "generation_parameters"

## 💡 Tips & Trik

### ✅ DO (Lakukan)
- Upload foto produk dengan **resolusi tinggi**
- Gunakan foto dengan **background bersih** untuk hasil terbaik
- Coba **semua 5 variasi** untuk mendapatkan hasil terbaik
- Sesuaikan **lighting dan atmosphere** dengan brand Anda
- Tambahkan **instruksi custom** yang spesifik

### ❌ DON'T (Jangan)
- Upload foto blur atau resolusi rendah
- Gunakan foto dengan watermark besar
- Harapkan hasil instan (AI butuh waktu proses)
- Lupa save prompt JSON yang sudah di-generate

## 🎯 Keunggulan Sistem Ini

### 1. **Produk 100% Identik**
System instruction memastikan AI **tidak mengubah** model, bentuk, motif, atau warna produk. Hanya background dan lighting yang berubah.

### 2. **5 Variasi Otomatis**
Tidak perlu berpikir tentang background - sistem sudah menyediakan 5 variasi profesional yang berbeda.

### 3. **Kualitas 4K**
Semua prompt dirancang untuk menghasilkan gambar **resolusi 4K** dengan tone warna tajam, terang, dan jelas.

### 4. **Siap Pakai untuk Promosi**
Hasil foto cocok untuk:
- Instagram feed & story
- Facebook ads
- Marketplace (Tokopedia, Shopee, dll)
- Website e-commerce
- Banner promosi
- Katalog digital

### 5. **Multi-Platform**
Satu prompt bisa digunakan di berbagai AI generator (Grok, ChatGPT, Gemini, dll).

## 🔧 Troubleshooting

### Masalah: "Harap unggah gambar terlebih dahulu!"
**Solusi**: Pastikan Anda sudah upload gambar produk sebelum klik generate.

### Masalah: "Minimal 2 gambar diperlukan untuk digabungkan!"
**Solusi**: Mode "Gabungkan Gambar" memerlukan minimal 2 gambar. Upload lebih banyak gambar.

### Masalah: Hasil AI tidak sesuai harapan
**Solusi**: 
- Coba variasi lain (dari 5 yang tersedia)
- Tambahkan instruksi custom yang lebih spesifik
- Gunakan foto produk dengan kualitas lebih baik
- Adjust lighting dan atmosphere settings

### Masalah: Produk berubah bentuk di hasil AI
**Solusi**: 
- Pastikan Anda paste **seluruh JSON prompt** (bukan hanya core_prompt)
- System instruction sangat penting untuk menjaga identitas produk
- Upload gambar referensi yang jelas ke AI generator
